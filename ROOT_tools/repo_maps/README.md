# Repository Maps

These files exist so connector-based assistants can navigate this repository without GitHub code search.

- `java_file_index.txt` - Java source paths only.
- `source_file_index.txt` - source/config/docs/text-like paths, excluding diagnostics and binary assets.
- `repo_file_index_full.txt` - all tracked paths; may be too large for some connector fetches.
- `chunks/` - full tracked path index split into connector-readable chunks.

Use `java_file_index.txt` first for code work.
