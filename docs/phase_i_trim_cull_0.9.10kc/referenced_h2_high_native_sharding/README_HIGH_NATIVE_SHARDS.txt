THE MECHANIST 0.9.10kc — HIGH_NATIVE GENERATED ART PAYLOAD SHARD

This is one shard of the high/native external art package.

Install/use:
1. Create one folder for the high/native payload, for example:
   TheMechanist_generated_art_high_native_0.9.10kc/

2. Extract ALL FOUR high_native shard zip files into that same folder.
   Let the folders merge. Do not place each shard in its own separate final folder.

3. After extraction, the folder should contain:
   assets/graphics/generated/high_native/<sheet>/<png files>

4. In The Mechanist launcher/options:
   PAYLOAD = the folder you extracted into
   ART QUALITY = high_native

Low_32 remains bundled inside the core game. The high/native tier is intentionally external and sharded because the single high/native zip was too large to surface reliably.
